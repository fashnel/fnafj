# FNaF Java Edition

![Menu picture](https://github.com/fashnel/fnafj/blob/master/assets/menu/menu.jpg)

## 📄 Описание
2D хоррор-игра в жанре Survival Horror, написанная на чистом Java. Проект вдохновлен вселенной Five Nights at Freddy's.
Это мой личный проект, нацеленный на изучение объектно-ориентированного программирования и работы с графическими библиотеками Java.

## ✨ Особенности
- **Core Gameplay:** Реализована механика камер, ограниченной энергии и взаимодействия с дверьми;
- **AI:** Простая логика поведения противников (аниматроников);
- **Engine:** Написано без использованя сторонниих игровых движков.

## 🛠 Стек
- **Язык:** Java (JDK 17);
- **Графика:** Swing.

## 🏗 Разработка
В ходе проекта я решал задачи:
- Оптимизации циклов отрисовки (Game Loop);
- Управления ресурсами (аудио, изображения);
- Реализации логики состояний игры (Game States).

Управление: 
Ecs — Выход в гланой меню
Space — Доступ к системе видеонаблюдения 
1-6 — Переключение камер
Q/E — Левая и правая дверь
